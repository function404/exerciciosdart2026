# Atividade 11/03/2026 - Práticas Aplicadas

## 📝 Sobre a Atividade

Conjunto de 4 desafios práticos que consolidam os conhecimentos em modelagem de classes, herança, consumo de dados externos via protocolo HTTP, manipulação de dados em tempo real e segurança rigorosa de tipagem.

## 📂 Estrutura dos Exercícios

- **`exercicioBiblioteca.dart`**: Implementação de um sistema de acervo utilizando classes abstratas e herança para diferenciar o cadastro e as propriedades de Livros e DVDs.
- **`exercicioClimaApi.dart`**: Consumo real da API pública do _Open-Meteo_ utilizando o pacote `http`, incorporando tratamento de resposta do servidor e conversão/extração de dados de um formato JSON.
- **`exercicioChat.dart`**: Simulação de um aplicativo de chat em tempo real utilizando tipagem `Stream`, `async*` e a palavra-chave `yield` para a entrega compassada de uma lista de mensagens.
- **`exercicioNullSafety.dart`**: Refatoração e estruturação de classes aplicando corretamente os operadores de segurança de nulos nativos do Dart (`?`, `??`, `late`).
